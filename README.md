# Multi-Container Todo App (Node.js + MongoDB)

This project is a simple **multi-container application** using Docker Compose.  
It consists of:

- **Node.js API** (Express + Mongoose) for a Todo List
- **MongoDB** database
- Optional **Nginx reverse proxy** (bonus)
